## ✨ المميزات

- 🔐 **صفحة تسجيل الدخول** — تحقق كامل من الحقول مع إظهار/إخفاء كلمة المرور
- 📝 **استمارة تسجيل الطلاب** — استمارة متكاملة مع شريط تقدم تلقائي
- 🏫 **صفحة مميزات المدرسة** — عرض احترافي للأقسام والإحصائيات
- 📱 **متجاوب مع الموبايل** — يعمل على جميع أحجام الشاشات
- 🌐 **دعم كامل للعربية** — RTL، خطوط عربية، تصميم من اليمين لليسار
- ✅ **تحقق من المدخلات** — JavaScript خالص بدون مكتبات خارجية

## 🗂️ هيكل المشروع

```
iraqi-school-website/
│
├── 📄 sign_in.html          ← صفحة تسجيل الدخول والتسجيل الأولي
├── 🎨 sign_in.css           ← تصميم صفحة الدخول (بطاقة ثلاثية الأبعاد)
│
├── 📄 form.html             ← استمارة تسجيل الطلاب
├── 🎨 form.css              ← تصميم الاستمارة مع شريط التقدم
│
├── 📄 iraqi_school.html     ← الصفحة الرئيسية للمدرسة
├── 🎨 iraqi_school.css      ← تصميم الصفحة الرئيسية
│
├── 📄 advantages.html       ← صفحة مميزات المدرسة
├── 🎨 advantages.css        ← تصميم صفحة المميزات
│
└── 📁 images/               ← صور المدرسة
```

## 🚀 تشغيل المشروع

```bash
# 1. استنسخ المستودع
git clone https://github.com/S-mohannad/front_end-_iraqi-school-website.git

# 2. افتح المجلد
cd front_end-_iraqi-school-website

# 3. افتح sign_in.html في المتصفح
```

> لا توجد تبعيات أو متطلبات تثبيت — HTML/CSS/JS خالص ✅
